<p>To avoid tedious tasks like regularly changing one's password, this app allows the user to change it directly on their mobile phone with one click. This results not only in less effort for the individual user, but also in more secure passwords. These are in fact randomly generated according to the criteria set by the UCS.</p><p><br><b>Features:</b></p><ul><li>Changing the password with the click of a button</li><li>Showing the history of set passwords</li><li>Changing the password generation<ul><li>By length</li><li>To only contain characters which can be used in a url without escaping</li></ul></li><li>Protection of saved data (see <a href='https://github.com/CodeSpoof/uniPass#security-and-limitations' target='_blank' rel='nofollow noopener'>Security and Limitations</a>)</li></ul>
